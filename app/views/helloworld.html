A hello world attribute

Changes the behavior of any html element to popup an "hello world" alert when clicked...

javascript: 
app.directive('helloWorld2', function(){
    return {
        name: 'helloWorld2',
        restrict: 'A',        
        link: function(scope, element, attrs) {
            // alert on click...
            element.bind("click", function() {
                alert("HELLO WORLD!!!!");
            });
        }
    };
});
html: 
<p hello-world2>Click me if you dare!</p>

<img src="images/dog.jpg" hello-world2 />

result:

Click me if you dare!
